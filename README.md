# 🎓 School AI Assistant

Aik intelligent chatbot jo students aur parents ke school se related sawalon ka jawab deta hai.

## ✨ Features

- 🤖 AI-powered conversations
- 📚 School information (admissions, fees, timings)
- 👨‍🏫 Teacher aur subject details
- 📅 Exam aur holiday schedule
- 📝 Homework help
- 💾 Chat history saving
- 🔐 Admin dashboard

## 🛠️ Tech Stack

- **Backend:** Python + Flask
- **Frontend:** HTML + CSS + JavaScript
- **Database:** SQLite
- **AI:** OpenAI API

## 🚀 Installation

1. Clone the repository
```bash
   git clone https://github.com/yourusername/school-ai-assistant.git
   cd school-ai-assistant
```

2. Create virtual environment
```bash
   python -m venv chart
   chart\Scripts\activate
```

3. Install dependencies
```bash
   pip install -r requirements.txt
```

4. Set up environment variables
   - Create a `.env` file
   - Add your OpenAI API key

5. Run the application
```bash
   python app.py
```
